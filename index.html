Welcome to the Golem wiki!

Current Status

The Golem software is in Alpha testing. Users can install the software on a variety of platforms and participate in testing. The first use of Golem, called Brass Golem, will provide CGI rendering for Blender. The Golem project milestones do not have set dates, Brass Golem is expected to be completed in 2018. To follow the progress of Brass Golem, you can follow our Kanban

Golem is open source and distributed under GPLv3 license.

Vision

With Golem, anyone will be able to rent their unused computing resources. Simultaneously, Golem will enable any user to buy computing time from other users to complete virtually any computationally demanding task. Consequently, Golem will create the first global market for idle computing power. Golem is a decentralized P2P network that uses Ethereum and smart contracts for transactions.
